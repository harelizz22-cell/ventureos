# Scripts

Parent: Repository root.

## Purpose

This folder is reserved for project scripts that support approved development, documentation, validation, maintenance, or operational workflows.

## What Belongs Here

- Approved helper scripts.
- Repository maintenance scripts.
- Documentation validation scripts.
- Development workflow automation.

## What Does Not Belong Here

- Production application code.
- Unapproved runtime automation.
- Secrets, credentials, or environment-specific values.
- Architecture decisions or governance policy definitions.

## Current Status

Reserved for future implementation.
