# Archive

Parent: Repository root.

## Purpose

This folder is reserved for retired, superseded, or historical project materials that must be preserved without remaining active source-of-truth content.

## What Belongs Here

- Superseded reference material.
- Historical drafts retained for context.
- Retired planning artifacts.
- Archived review material approved for retention.

## What Does Not Belong Here

- Active source-of-truth documentation.
- Production application code.
- Current phase specifications.
- Secrets, credentials, or environment-specific values.

## Current Status

Reserved for future implementation.
