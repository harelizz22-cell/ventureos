# ADR-014: Value Graph

Parent: `docs/adr/README.md`

Source Of Truth: This file is the source of truth for the accepted Value Graph decision.

## Status

Accepted

## Context

VentureOS must eventually explain why value was created, not only what happened.

## Decision

VentureOS adopts Value Graph architecture.

The Value Graph connects Founder Decision -> Workflow -> Capability -> Execution -> Cost -> Revenue -> Profit -> Enterprise Value.

## Consequences

- Value creation becomes traceable across decisions, execution, cost, and outcomes.
- Founder Decision Graph and Business Intelligence depend on value traceability.
- Future analytics must connect technical action to business value.

## Enforcement

Future capabilities and workflows must preserve traceability to measurable business outcomes where meaningful.
