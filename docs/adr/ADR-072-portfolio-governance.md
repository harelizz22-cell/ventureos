# ADR-072: Portfolio Governance

Status: Accepted

Date: 2026-07-01

## Context

VentureOS is Portfolio-first and Enterprise Value-first. Capital decisions must account for concentration, diversification, liquidity, follow-on planning, and correlated risk.

## Decision

Adopt Portfolio Governance as the architecture concern for portfolio diversification, sector exposure, capital concentration, risk balancing, liquidity planning, expected return, portfolio health, and follow-on planning.

## Rationale

Single-venture optimization can damage long-term Enterprise Value when portfolio risk, liquidity, and follow-on needs are ignored.

## Consequences

- Portfolio Governance informs major allocation, release, reallocation, and recovery decisions.
- Expected return must distinguish assumptions, evidence, confidence, and time horizon.
- Portfolio health must include financial and Enterprise Value signals.

## Enforcement

Material allocation decisions must consider portfolio exposure, risk, liquidity, expected return, and Enterprise Value impact before approval.
