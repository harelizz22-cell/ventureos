# Phase 0: Governance Foundation

## Objective

Define the enforceable governance model for assisted operation.

## Scope

- Policy Engine rule categories.
- Approval model.
- Decision and evidence recording requirements.
- Agent lifecycle controls.
- Tool Gateway and AI Gateway access rules.

## Exit Criteria

- Governance rules are reviewable and testable.
- Approval thresholds are specified.
- Required audit records are defined.
- Implementation scope for governance services is ready for ADR review.
