# Typography

Parent: `docs/design/README.md`

Source Of Truth: This file is the source of truth for VentureOS typography direction.

## Direction

Use readable enterprise typography. Labels, headings, and body text should feel precise, calm, and legible.

## Rules

- Avoid excessive terminal style.
- Use mono font only for IDs, timestamps, hashes, logs, and technical identifiers.
- Normal product labels should use readable title case.
- Avoid excessive ALL CAPS.
- Use hierarchy through size, weight, spacing, and placement rather than decorative treatment.

## Product Language

Prefer:

- Venture.
- Operation.
- Decision.
- Workflow.
- Evidence.
- Governance.

Avoid using "Mission" as the primary product term.
