# Color System

Parent: `docs/design/README.md`

Source Of Truth: This file is the source of truth for VentureOS color direction.

## Initial Direction

- Base dark graphite background.
- Deep neutral surfaces.
- One controlled signature accent.
- Semantic colors only for state.
- No decorative color usage.

## Semantic Meanings

- Approved.
- Needs Founder Attention.
- Governance Blocked.
- Information.
- Neutral / Unknown.

## Rules

- Semantic colors must map to state, not mood.
- Accent color must be controlled and sparse.
- Avoid full-screen gradients, decorative glow, neon, and high-saturation effects.
- Neutral surfaces should carry most of the interface.
- Error and blocked states must be distinguishable from attention states.

## Open Decisions

- Exact palette values.
- Signature accent color.
- Contrast targets for final UI implementation.
