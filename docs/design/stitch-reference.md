# Stitch Reference

Parent: `docs/design/README.md`

Source Of Truth: This file is the source of truth for how Stitch exploration may be used.

## Purpose

Define the status of Stitch exploration in the VentureOS design process.

## Rule

Stitch is exploration only. It is visual inspiration, not final implementation and not source of truth.

## Allowed Use

- Identify useful visual mood.
- Explore layout possibilities.
- Compare interface density.
- Inform discussion about tone.

## Not Allowed

- Treat Stitch output as approved UI.
- Copy implementation details without review.
- Override `docs/design/`.
- Introduce product concepts not approved by architecture or founder decision.

## Approved Direction From Exploration

- Dark professional operating interface.
- Founder Operating Console as main entry concept.
- Founder Command Center as primary screen.
- Calm, premium, precise, executive tone.
- Enterprise operating-system posture rather than AI novelty dashboard.
