# Visual Identity

Parent: `docs/design/README.md`

Source Of Truth: This file is the source of truth for VentureOS visual identity direction.

## Identity Direction

VentureOS should feel like a browser-first enterprise operating system for founder governance and venture operation.

## Tone

- Calm.
- Premium.
- Precise.
- Executive.
- Focused.
- Trustworthy.

## Interface Character

The interface should communicate control, evidence, and operational readiness. It should not imply entertainment, speculation, or agent novelty.

## Primary Product Entry

The Founder Operating Console is the main entry concept. The Founder Command Center is the primary screen inside the console and should make founder status, decisions, and operations immediately legible.

## Visual Restraint

Visual identity should use restraint. Color, motion, contrast, and emphasis exist to communicate system meaning.
