## Purpose

`docs/design/` defines the official VentureOS design direction, design language, interaction expectations, and product interface rules.

## Scope

This folder covers design principles, visual identity, color, typography, layout, components, interaction, accessibility, and references from exploration work.

## Design Source Rule

The current Stitch exploration is visual inspiration only. It is not final implementation and is not the source of truth.

The official design source of truth is `docs/design/`.

## Product Direction

- Browser-first enterprise operating system.
- Desktop is primary.
- Mobile is companion only.
- Dark professional interface.
- Founder Command Center is the main entry concept.
- Calm, premium, precise, executive.
- Avoid generic AI dashboard patterns.
- Avoid playful, neon, cyberpunk, and glassmorphism styling.
