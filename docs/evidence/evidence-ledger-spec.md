# Evidence Ledger Specification

## Purpose

The Evidence Ledger records material used to support or reject actions, opportunities, and decisions.

## Required Fields

- Evidence ID
- Title
- Source
- Source type
- Summary
- Confidence
- Related opportunity
- Related decision
- Created by
- Created at
- Review status

## Rules

- Evidence must be linked to meaningful decisions.
- Evidence quality and confidence must be explicit.
- Unsupported claims must remain unvalidated.
