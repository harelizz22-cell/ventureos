# Capability Registry

Parent: `docs/architecture/ventureos-architecture.md`

Source Of Truth: This file is the source of truth for Capability Registry architecture.

## Purpose

The Capability Registry defines system capabilities independently from specific agents.

## Required Capability Attributes

- Purpose.
- Risk class.
- Dependencies.
- Required approvals.
- Allowed tools.

## Rule

Capabilities do not grant execution authority by themselves.
