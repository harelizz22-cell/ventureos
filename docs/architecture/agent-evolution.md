# Agent Evolution

Parent: `docs/architecture/ventureos-architecture.md`

Source Of Truth: This file is the source of truth for agent evolution architecture.

## Lifecycle

Capability Gap Detected -> Agent Proposal -> Risk Review -> Auditor Review -> Founder Approval -> Sandbox Test -> Activation Approval

## Rule

The system may recommend and draft a new agent contract. It may not activate a new agent without founder approval.
