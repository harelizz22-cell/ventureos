# Investment Dossier

Parent: `docs/architecture/ventureos-architecture.md`

Source Of Truth: This file is the source of truth for Investment Dossier architecture.

## Purpose

The Investment Dossier defines the future investor-facing package for a Venture opportunity.

It explains the opportunity, evidence, assumptions, risks, capital need, governance model, and compliance status in a structured form suitable for review.

## Required Contents

An Investment Dossier must include:

- Problem
- Solution
- Market
- Why now
- Opportunity score
- Thesis alignment
- Evidence summary
- Risk summary
- Required capital
- Use of funds
- Milestone plan
- Expected outcomes
- Expected ROI range
- Downside risks
- Kill criteria
- Governance model
- Team / execution model
- Legal/compliance status
- Investor suitability notes

## Evidence Distinction

The dossier must clearly distinguish:

- Assumptions
- Evidence
- Forecasts
- Confirmed facts

## Rule

No Investment Dossier may use guaranteed return language.

The dossier is an investor-facing explanation package. It does not sell securities, process investments, approve investors, or authorize capital movement.
