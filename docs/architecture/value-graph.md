# Value Graph

Parent: `docs/architecture/ventureos-architecture.md`

Source Of Truth: This file is the source of truth for Value Graph architecture.

## Purpose

The Value Graph connects Founder decisions, workflows, capabilities, execution, cost, revenue, profit, and enterprise value.

## Graph Path

Founder Decision
-> Workflow
-> Capability
-> Execution
-> Cost
-> Revenue
-> Profit
-> Enterprise Value

Future funding paths may add Funding Round, Investor Commitment, Milestone Capital Release, and Outcome Tracking nodes where legally/compliance approved.

## Rule

VentureOS must eventually explain why value was created, not only what happened.

The Value Graph must not treat forecasts or expected ROI as confirmed value.

## Placeholder Status

This is an architecture placeholder. It does not define graph storage, algorithms, UI, analytics models, or implementation.
