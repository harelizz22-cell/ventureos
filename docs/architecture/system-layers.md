# System Layers

Parent: `docs/architecture/ventureos-architecture.md`

Source Of Truth: This file is the source of truth for VentureOS system layers.

## Layers

1. Founder Layer.
2. Organization Layer.
3. Portfolio Layer.
4. Venture Layer.
5. Domain Layer.
6. Capability Layer.
7. Execution Layer.
8. Evidence, Decision, and Audit Layer.
9. Resource and Asset Layer.
10. Recovery Layer.

## Operating Hierarchy

Founder -> Organization -> Portfolio -> Venture -> Domain -> Capability -> Workflow -> Execution -> Evidence -> Decision -> Audit

## Rule

New components must be placed in one layer and must not bypass layers above them.
