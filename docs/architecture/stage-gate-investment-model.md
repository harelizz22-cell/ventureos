# Stage Gate Investment Model

Parent: `docs/architecture/ventureos-architecture.md`

Source Of Truth: This file is the source of truth for Stage Gate Investment Model architecture.

## Purpose

The Stage Gate Investment Model defines mandatory investment gates for Venture lifecycle progression.

## Venture Lifecycle

Every Venture must support:

- Idea
- Validation
- Prototype
- MVP
- Revenue
- Growth
- Scale
- Mature
- Exit
- Archive

Investment rules differ at every stage.

## Mandatory Investment Gates

- Stage 0: Idea. Maximum investment: Founder defined.
- Stage 1: Validation.
- Stage 2: Prototype.
- Stage 3: MVP.
- Stage 4: Revenue.
- Stage 5: Growth.
- Stage 6: Scale.
- Stage 7: Mature.
- Stage 8: Exit.
- Stage 9: Archive.

Each transition requires:

- Evidence
- Expected ROI
- Risk score
- Governance approval
- Capital recommendation

## Rule

No stage transition may bypass evidence, risk scoring, governance approval, and capital recommendation requirements.
