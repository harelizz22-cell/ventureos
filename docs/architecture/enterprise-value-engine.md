# Enterprise Value Engine

Parent: `docs/architecture/ventureos-architecture.md`

Source Of Truth: This file is the source of truth for Enterprise Value Engine architecture.

## Purpose

The Enterprise Value Engine evaluates long-term enterprise value across Organizations, Portfolios, and Ventures.

Enterprise Value is the primary optimization objective of VentureOS.

## Evaluates

- Revenue
- Profit
- Assets
- Knowledge
- Brand
- Customer base
- Recurring revenue
- Strategic positioning
- Operational maturity
- Future growth potential
- Thesis alignment
- Hypothesis validation quality
- Portfolio diversification
- Evidence-backed learning

## Rule

Revenue is an important KPI. Enterprise Value is the primary objective.

Every subsystem ultimately exists to increase Enterprise Value while preserving Founder control, governance, transparency, and capital discipline.

Thesis Engine, Hypothesis Engine, Portfolio Diversification, and Learning Engine support Enterprise Value evaluation. They do not replace governance, Founder approval, or the no autonomous money movement rule.
