# Opportunity State Machine

## States

- Idea
- Evidence Required
- Researching
- Validation Proposed
- Validation Approved
- Validation Running
- Validation Complete
- Build Proposed
- Build Approved
- Launched
- Operating
- Paused
- Rejected

## Rules

- Build cannot precede validation.
- Approval is required before meaningful external action in Assisted Mode.
- Evidence and decision links are required for promotion beyond validation stages.
