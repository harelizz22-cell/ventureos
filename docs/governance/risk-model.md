# Risk Model

Parent: `docs/project/ventureos-system-bible.md`

Source Of Truth: This file is the source of truth for governance risk categories.

## Purpose

The Risk Model classifies proposed actions so review requirements can be applied consistently.

## Risk Categories

- Architecture risk.
- Governance risk.
- Security risk.
- Financial risk.
- External integration risk.
- Production asset risk.
- Autonomy expansion risk.

## Rule

High-risk changes require risk review before founder approval.
