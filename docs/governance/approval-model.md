# Approval Model

Parent: `docs/project/ventureos-system-bible.md`

Source Of Truth: This file is the source of truth for approval model rules.

## Approval Principles

- Founder approval is required for meaningful external action in Assisted Mode.
- Agents cannot approve their own work.
- Risk-sensitive actions require Risk Agent review.
- Governance-sensitive actions require Auditor Agent review.
- Spending above configured thresholds requires approval.

## Approval Records

Each approval record must include requester, approver, action, evidence, decision link, timestamp, expiration if applicable, and resulting state transition.
