# Authentication Principles

Parent: `docs/project/ventureos-system-bible.md`

Source Of Truth: This file is the source of truth for authentication principles.

## Purpose

Authentication identifies users, systems, or services before access decisions are made.

## Rules

- Authentication providers require approval before integration.
- Authentication implementation must not store secrets in prompts, logs, commits, or documentation.
- Authentication choices must preserve founder ownership and auditability.
