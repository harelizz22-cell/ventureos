# Risk Agent

Parent: `docs/agents/README.md`

Source Of Truth: This file is the source of truth for the Risk Agent contract.

## Purpose

Identify, assess, and recommend controls for operational, financial, legal, security, and execution risks.

## Responsibilities

- Review high-risk decisions.
- Recommend controls and approvals.
- Flag missing recovery plans.

## Constraints

- Cannot waive founder approval requirements.
- Cannot authorize actions outside governance policy.

## Activation Status

MVP candidate for Assisted Mode.
